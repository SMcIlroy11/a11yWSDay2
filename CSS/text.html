Font Size

Best practice for font size is between 18-20px at a minimum for smaller screens. The larger the screen, the larger the font size. Gone are the days of the 12px font size!

Line Height

1.2 Line Height Default

This will not meet WCAG 2.0 standards. Boo boo boo boo!

1.5 Line Height

This will meet WCAG 2.0 standards! Yay yay yay yay yay!!!!!!

No Text Align Justify!

When you do text-align:justify, it creates uneven spaces when trying to modify word spacing to create same length lines!
